# DCTL-creations
This repository is used as a list of my own DCTL's, created for easy workflow in DaVinci Resolve

# Exposure Control.dctl - Change exposure stops with a simple checkbox
This DCTL allows you to easily adjust the exposure of a clip from +5 to -5 stops in DaVinci Wide Gamut.
Inside the DCTL, the color space gets changed to linear for an easier transformation within the DCTL coding language, but then outputs the result again in DWG/WI.
Personally, I use this tool to quickly create exposure adjustments for Dailies creation.
